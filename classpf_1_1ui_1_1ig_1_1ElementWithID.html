Detailed Description

Interface for all ImGui elements with IDs.

Adds functionality of pushing unique names to ImGui render tree.

Warning
If there are duplicate IDs in the render tree the program may have some undefined behavior for functions which use IDs.
See also
Renderable
Attention
For classes which don't need IDs use ElementBaase
For classes with hover/focus/tooltip etc. use ItemElement
