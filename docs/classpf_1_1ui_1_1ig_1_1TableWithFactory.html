Detailed Description

template<typename Factory, std::derived_from< Renderable >... Cells>
class pf::ui::ig::TableWithFactory< Factory, Cells >

Table, which can contain any Renderable as a cell value.

Table width is statically sized. Each column should contain the same type of element, thus Cells... are used.

Template Parameters
Factoryfactory for utility method addRow(Factory::CreationType), if factory is CommonPlaceholder, then no factory is used
Cellstypes of Renderables used for each column
