Detailed Description

Root class of an entire UI tree. Provides an ability to create windows and dialogs.

The class has it's own menu bar, which is applied to the entire render window.

It also takes care of config saving for Savable elements.

Attention
: For actual use it needs to be inherited from and render_impl should be reimplemented with your own rendering logic. You also need to update fonts when required based on variable shouldUpdateFontAtlas.
