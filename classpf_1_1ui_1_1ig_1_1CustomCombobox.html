Detailed Description

template<typename T, std::derived_from< Renderable > R>
class pf::ui::ig::CustomCombobox< T, R >

Combobox where rows can be whatever is provided by the user.

Rows are created using a factory from stored value.

Template Parameters
Ttype stored in each row
Rtype stored in each row
